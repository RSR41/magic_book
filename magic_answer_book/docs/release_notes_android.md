# Android Release Notes (Play Internal/Production)

## Version
- 0.1.3 (versionCode 2)

## 주요 변경사항
- **광고 제거 인앱결제(IAP) 추가**: 설정 화면에서 광고 제거 상품 구매 및 복원 경로 제공.
- **다시 뽑기 누적 광고 정책 적용**: 재시도 흐름에서 광고 노출 타이밍/빈도 정책을 정비.
- **광고 정책 고지 업데이트**: 광고 노출 조건 및 사용자 안내를 정책 친화적으로 개선.
- **면책 고지 강화**: 앱 답변은 참고·오락용이며 중요한 의사결정 근거로 사용하지 않도록 안내.

## 리뷰어 확인 포인트
- 결제 성공/실패/취소 시 상태 전환이 정상인지.
- 재설치 이후 복원 경로에서 구매 상태 재동기화가 정상인지.
