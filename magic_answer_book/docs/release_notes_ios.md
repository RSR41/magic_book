# iOS Release Notes (TestFlight/App Store)

## Version
- 0.1.3 (Build 2)

## 주요 변경사항
- **광고 제거 인앱결제(IAP) 추가**: 설정 화면에서 광고 제거 상품 구매 및 복원 지원.
- **다시 뽑기 누적 광고 정책 적용**: 반복 사용 시 정책에 맞게 보상/전면 광고 노출 로직 정비.
- **광고 정책 문구 정리**: 앱 내 광고 노출 기준과 사용자 안내 문구를 최신 정책 기준으로 업데이트.
- **면책 문구 강화**: 결과는 오락 목적이며 의사결정 책임이 사용자에게 있음을 명확히 고지.

## 리뷰어 확인 포인트
- 광고 제거 구매 후 광고 미노출 상태가 즉시 반영되는지.
- 앱 삭제/재설치 후 **Restore Purchases** 동작으로 광고 제거 상태가 복원되는지.
